# JitpackAndroidUtils
## Utilities for android projects

## How to use:

In your project's root build.gradle file

```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
  
In your modules build.gradle file  

```
  dependencies {
	        implementation 'com.github.sjd753:JitpackAndroidUtils:v1.0'
	}
```
